Streamer Homepage – Rhowyen

A custom-designed homepage for a live streamer focused on cozy, forest-themed aesthetics and community interaction.

The Goal

Create a personalized homepage that centralizes streaming links, community spaces, and live status while maintaining a cohesive visual identity.

Tech Stack

  • HTML5
  • Tailwind CSS (responsive, mobile-first)
  • Vanilla JavaScript
  • GitHub Pages deployment

Challenges & Solutions

Balancing personality-driven design with usability required careful spacing, collapsible sections, and clear hierarchy — especially on mobile screens.
