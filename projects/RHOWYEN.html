Streamer Homepage – Rhowyen

A custom-designed homepage for a live streamer focused on cozy, forest-themed aesthetics and community interaction.

The Goal

Build a centralized, personality-driven homepage that highlights streaming links, community spaces, and live status while remaining usable and readable on mobile devices.

Tech Stack

  • HTML5
  • Tailwind CSS (mobile-first layout)
  • Vanilla JavaScript
  • GitHub Pages deployment

Challenges & Solutions

Balancing heavy personality and themed visuals with usability required collapsible sections, clear hierarchy, and careful spacing—especially on mobile screens.
