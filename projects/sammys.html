Sammy’s Transportation Website

This project involved developing and refining a professional website for Sammy’s Transportation, a family-owned transportation service operating in Kiawah Island, Seabrook Island, and the greater Charleston area. The goal was to create a modern online presence that clearly communicates available services while allowing customers to conveniently book rides directly through the website.

Problem

The business needed a website that clearly presented its services and long-standing reputation while making it easier for customers to request transportation. Traditional booking through phone calls alone created friction for users who preferred a quick online scheduling option. Additionally, the site needed to maintain a professional appearance that would build trust with visitors searching for reliable transportation.

Solution

I implemented a WordPress-based website that highlights the company’s services, experience, and customer satisfaction. The site integrates a booking plugin that allows visitors to schedule rides directly through the website. I also customized sections of the design using HTML, CSS, and small JavaScript adjustments to ensure the booking system and site layout created a seamless and intuitive user experience.

Tech Stack

  • WordPress
  • HTML5
  • CSS
  • JavaScript
  • WordPress Booking Plugin

Key Takeaways

  • Experience customizing a WordPress site using HTML, CSS, and JavaScript
  • Integration of a third-party booking plugin for direct ride scheduling
  • Improving user experience through clear service sections and calls to action
  • Designing a professional layout that reflects a reliable service brand
